## 2025-05-15 - Redundant GSP Lookup Optimization **Learning:** Internal helper `get_meter_details` was performing an expensive API call to retrieve the Grid Supply Point (GSP) even when only MPAN/Serial identification was needed. **Action:** Introduced `include_gsp` flag (default `TRUE`) in `get_meter_details` to allow callers like `get_consumption` to skip redundant secondary API calls, reducing network overhead by one request per call.
